# HR_Dashboard

This project leverages Power BI to analyze workforce data and deliver clear, actionable insights into employee demographics, performance, engagement, and retention.

The dashboard is designed to support HR leaders and management in making data-driven workforce decisions.

---

### Project Objective
The object of this project is to:
- Analyze workforce composition and demographics
- Track employee performance, engagement, and absenteeism
- Monitor employee turnover and retention trends
- Provide visibility into department and manager-level performance

---

### Tools & Skills Applied
- Power BI
- DAX for KPI calculations
- Power Query for data transformation
- Interactive visuals and slicers
- HR analytics and reporting best practices

---

### Key Insights
- Clear visibility into workforce demographics and diversity
- Identification of departments with high turnover or absenteeism
- Relationship between performance, engagement, and retention
- Managerial impact on employee outcomes
